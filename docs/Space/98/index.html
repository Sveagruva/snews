Роскосмос запатентовал ловушку для космического мусора

08.06.20

Роспатент выдала Роскосмосу патент на устройство для очистки космоса. В описании патента написано следующее:

Изобретение относится к космической технике и может быть использовано для очистки околоземного космического пространства от относительно крупного по размеру космического мусора, такого как прекратившие активное существование космические аппараты, разгонные блоки, последние ступени ракет.

В документе устройство описывается как сеть в форме конуса, основание которой закреплено раздвижной рамой. Основание данной сети должно крепится на раздвижной раме и иметь двигатели для маневрирования. В конце данной сети-сачка будет располагаться космический корабль.

Работа данной ловушки описывается следующим образом:

  1. С помощью двигателей на раме "сачка" сеть захватывает космический мусор.
  2. Рама сжимается тем самым закрывая единственный выход из сети.
  3. Последним этапом пойманный мусор, либо буксируется и сгорает в атмосфере, либо переводится на орбиту захоронения.
