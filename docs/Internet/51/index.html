32-битная эпоха офицально закончилась

18.05.20

На данный момент поддерживается как 32-битная так и 64-битная windows 10. 32-битная система имела смысл, когда windows 10 пришла на замену windows 7. Уже сейчас тяжело найти новый компьютер с 32-битной windows на борту, но теперь Microsoft хочет сделать такое невозможным.

Как сообщает Neowin, новые компьютеры с windows 10 2004 будут поставляться только с 64-битной версией. Компания продолжит поддержку и обновления уже установленных 32-битных windows, если конечно компания резко не сменит политику. Первым шагом который она сделала к прекращению 32-битной эпохи было увеличение минимального накопителя до 32 гигабайт, в прошлом году, для обоих версий windows.
