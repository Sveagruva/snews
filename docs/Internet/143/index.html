Microsoft выпустила программу для восстановления удалённых файлов

30.06.20

Программа с именем "Windows File Recovery" была выпущена в открытый бесплатный доступ в собственном магазине приложений.

Случайно удалили важный файл? Очистили жесткий диск? Не знаете, что делать с поврежденными данными? Windows File Recovery может восстновить ваши данные.

Удалённые из корзины файлы не удаляются с диска окончательно, а вместо этого их место помечается как свободное для записи. Но если данное место не было занято, то данные всё ещё можно считать, что и делает данная программа.

Программа поддерживает самые популярные файловые системы и типы файлов. К сожалению программа не для массового пользователя, так как использовать её можно только через командную строку. Для восстановления файлов обычным пользователям всё ещё необходимо искать программу с графическим интерфейсом.
