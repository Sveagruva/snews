Разработка TON прекращена

30.05.20

На сайте платформы было опубликовано следующие сообщение:

Оригинальная команда разработчиков TON прекращает свое активное участие в проекте TON по причинам, объясненным далее. Усилия команды были перенаправлены на другие проекты. Некоторые из внутренних ветвей разработки, которые были в активной разработке и тестировании, могут быть интегрированы в основную ветку. Некоторые мелкие исправления и ответы на вопросы Github могут иногда появляться, если у кого-либо из членов первоначальной команды есть свободное время и желание внести свой вклад в усилия сообщества. Принципиальная разработка кода TON передается сообществу свободного исходного кода.

Весь код проекта находится в открытом доступе и любой может его продолжить. Сеть уже работала в тестовом режиме, но её закрыли. Разработка TON была прекращена из-за запрета американского суда.
