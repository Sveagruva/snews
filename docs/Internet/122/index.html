Google Assistant в телевизорах научат узнавать пользователей по голосу

17.06.20

Голосовые ассистенты становятся всё умнее и умнее. Они растут также быстро, как и потребность в них. Один из них - Google Assistant, который встроен практически во все устройства с операционной системой андроид. Телевизоры не стали исключением.

Google Assistant для телевизоров стал уже привычной функцией. Его используют, чтобы что-то найти или ввести текст. Это намного удобнее, чем пользоваться стрелками на пульте и виртуальной клавиатурой.

Если большинство устройств способно распознавать человека по голосу и предлагать ответы в соответствии с его профилем, то телевизоры стали исключением. Как сообщает издание 9to5google, им удалось обнаружить в коде работу над функцией Voice Match для телевизоров.

Появление данной функции значительно расширяет возможности телевизоров. Например, если говорящий попросит открыть Netflix, то ассистент с помощью голоса поймёт какой профиль необходимо выбрать для данного пользователя и выберет его автоматически. Это также даёт возможность устанавливать ограничение по времени или какие-либо настройки для каждого человека по отдельности.
