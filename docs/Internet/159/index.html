На Android-смартфонах теперь есть возможность зашифровать важные файлы

06.08.20

В официальном приложении Google Files появилась возможность зашифровать важные файлы. Для доступа к файлам будет использоваться 4-значный PIN-код.

При сворачивании или закрытии приложения папка будет автоматически блокироваться для обеспечения большей безопасности.

Приложение также обладает другими функциями. Главная из них это удаление ненужных файлов. Программа автоматически анализирует файлы на телефоне и предлагает удалить дубликаты или просто тяжёлые файлы.

Также там есть встроенный файловый менеджер и сортировка файлов по категориям. В файловом менеджере доступна функция для быстрого резервного копирования на Google Drive.

Помимо этого приложение позволяет передавать файлы между устройствами без интернета.
