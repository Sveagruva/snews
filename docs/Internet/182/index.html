Соцсеть ВКонтакте хочет вернуть украинских пользователей

13.09.20

В мае 2017 года соцсеть ВКонтакте была заблокирована на территории Украины по причине проведения анти-украинской политики.

В «Mail.ru Group» решили не ждать окончания запрета, а предоставить пользователям встроенные в приложение инструменты для обхода блокировки.

Помимо этого для возвращения пользователей компания разослала письма бывшим украинским пользователям и рассказала в видеоролике о нововведениях в соцсети, которые украинские пользователи пропустили:

Для привлечения внимания ВК выпустила бесплатный набор стикеров. Специально для украинских пользователей в ленту новостей был добавлен фильтр для показа только новостей страны.
