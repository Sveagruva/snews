В Москве откроется KFC с автоматизированной системой выдачи

11.06.20

Сеть ресторанов KFC в скором времени откроет ресторан в Москве без касс и официантов. Заказы по-прежнему будут готовиться на кухне обычными поварами, а выдавать их будет роборука.

Помещённые на конвейер готовые заказы будут распределены роборукой в соответственные ячейки — камеры для хранения.

Для создания и оплаты заказа будут установленны привычные терминалы с сенсорным экраном, но не всё так просто. Терминалы будут оснащены системой распознавания лиц. Данная система даст возможность оплачивать покупки без карты. Помимо этого специальный алгоритм будет анализировать историю покупок и при следующем визите предлагать продукцию, которую пользователь вероятно захочет купить.

Сделать заказ можно не только в самом ресторане, но на пути к нему. Для этого KFC создала мобильное приложение.
