Opera выпустила крупное обновление для одноимённого браузера

22.06.20

Игровой браузер Opera GX получил крупное обновление, которое уже можно установить нажав на логотип браузера в левом верхнем углу и выбрав пункт с названием "обновление и восстановление".

Обновление приносит три новые крутые фичи, а именно: Hot Tabs Killer, Discord и Force Dark Mode. Помимо этого игровой браузер недавно унаследовал такую функцию как "рабочие пространства". Давно привычными стали социальные сети в боковой панели и поиск по открытым вкладкам.

Discord - голосовой чат преимущественно используемый геймерами, теперь внедрён в браузер.

Hot Tabs Killer - это диспетчер задач для вкладок. Теперь функция легкодоступна в боковой вкладке "GX Control". Она показывает потребление ресурсов процессора и оперативной памяти по вкладкам.

Force Dark Mode приносит тёмную тему на каждый сайт. Включить её можно в настройках браузера. На данный момент функция находится в бета-тестировании.
