Учёные создали умную одежду с использованием графена

21.06.20

Учёные из Манчестерского университета опубликовали своё новое изобретение в журнале Nano Letters. Их разработка представляет собой ткань из будущего, которая способна сохранять холод в жару и согревать в холод.

Принцип работы ткани основан на изменении пропускающей способности инфракрасного излучения с помощью графена. Данная группа учёных уже применяла подобную технологию, что бы скрыться от тепловизоров.

Для демонстрации разработки они создали прототип одежды. По их заявлениям применение данной технологии не ограничиться одной только одеждой. Большую востребовательность она получит в космосе.
