Huawei запатентовала дизайн смартфонов с подэкранными камерами

02.06.20

CNIPA опубликовала два патента Huawei 2019 года. В обоих патентах описываются подобные устройства главной особенностью является отсутствие какой-либо системы с фронтальной камерой. Из кнопок у устройств будет только кнопка включения-выключения сверху. Экран будет иметь сильные закругления.

Фронтальная камера не вырезана в дисплей. Также сверху устройства нет механизма для выдвижной камеры. Вряд ли компания запатентовала такие футуристичные дизайны для телефонов без фронтальной камеры. В версию с подэкраными камерами верится больше. Версию с подэкраными телефонами подтверждают утечки в начале года, о том что Huawei готовит технологию для выпуска во второй половине этого года.

Вероятно данный дизайн получит флагманская линейка Huawei Mate.
