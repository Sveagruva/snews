AI научили писать код по комментариям

06.07.20

Кевин Скотт и Сэм Альтман научили OpenAI писать код на Python при помощи открытых репозиториев на Github.

Меньше месяца назад компания искусственного интеллекта OpenAI представила свою третью версию алгоритма GPT.

Тогда компания продемонстрировала возможности алгоритма при правильном обучении. На данном примере алгоритм самостоятельно дополняет таблицу:

Работа алгоритма была продемонстрирована на ютубе. В видео показывается как натренированный алгоритм может писать код на Python. Код может быть продолжен с любого места, но для получения желаемого результата программист должен оставить комментарий с описанием действия функции. На самом деле даже комментарии не обязательны. Достаточно одного названия и входящих аргументов. В таком случае алгоритм сам напишет комментарий и код.

Для обучения были использованы примеры кода с открытых репозиториев на Github с использованием облачного суперкомпьютера Microsoft. Для написания кода AI может использовать уже написанные выше функции.
