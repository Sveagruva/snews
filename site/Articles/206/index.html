Как удалить с YouTube кнопку «воспроизвести на телевизоре»

16.12.20

Количество умных телевизоров постоянно растёт. Одной из функций данных устройств является наличия приложения YouTube, которое позволяет просматривать видео на телевизоре.

Помимо использования пульта приложение позволяет управление по Wi-Fi. Именно из-за этой функции при наличии телевизора в сети который вы подключены, ютуб плеер будет иметь кнопку для просмотра на телевизоре.

Данная кнопка не всегда является нужной, но её близкое расположение к кнопке перехода в полноэкранный режим сильно повышает шанс её случайного нажатия.

Для того что бы убрать данную кнопку можно использовать специальное расширение для Google Chrome. Данное расширение может быть легко отключено простым нажатием по иконке. После чего перезагрузка страницы вернёт удалённую кнопку.
