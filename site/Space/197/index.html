Nokia запустит сеть 4G на Луне

19.10.20

NASA в рамках программы Артемида выделила Nokia 14,1 миллиона долларов на строительство сети 4G LTE, которая в последствии будет переоборудована на 5G. Джеймс Рейтер считает, что создание мобильной связи является чрезвычайно важным для создания поселения на Луне.

При финансовой поддержке NASA, Nokia рассмотрит, как можно модифицировать наземные технологии для лунной среды, чтобы обеспечить надежную высокоскоростную связь.

Сотовая связь позволит не только быстрее координировать астронавтов, но и использовать оборудование для наблюдения и изучения Луны в режиме реального времени. Такие возможности будут доступны не только космонавтам. Для увеличения безопасности через созданную сеть можно будет вести прямую трансляцию на Землю.

Вдохновленная наземными технологиями, Nokia предлагает развернуть первую систему связи LTE / 4G в космосе. Система может поддерживать связь с лунной поверхностью на больших расстояниях, увеличенных скоростях и обеспечивать большую надежность, чем существующие стандарты.

Nokia уже пыталась в 2018 году запустить интернет на Луне для прямой трансляции видео высокого качества. Для этого она нашла партёров в лице Vodafone и PTScientists, но проект так и не был осуществлён.
