Исходный код Windows XP стал общедоступным

26.09.20

Исходный код операционной системы Windows XP был слит в сеть на форуме 4chan. Скачать его можно с помощью торрента. Помимо Windows XP в файлах торрента находится исходный код и других операционных систем:

  • MS-DOS 3.30 OEM Adaptation Kit;
  • MS-DOS 6.0;
  • DDKs / WDKs stretching from Win 3.11 to Windows 7;
  • Windows NT 3.5;
  • Windows NT 4;
  • Windows 2000;
  • Windows CE 3.0 Platform Builder;
  • Windows CE 4.2 Shared Source;
  • Windows CE 5.0 Shared Source;
  • Windows CE 6.0 R3 Shared Source;
  • Windows Embedded Compact 7.0 Shared Source;
  • Windows Embedded Compact 2013 (CE 8.0) Shared Source;
  • Windows 10 Shared Source Kit;
  • Windows Research Kernel 1.2;
  • Xbox Live;
  • Xbox OS;

Исходный код XP зашифрован и может быть расшифрован с помощью хэша ключа, который приложен в файле. От автора торрента:

Распространите этот код повсюду. Поскольку Microsoft утверждает, что любит открытый исходный код, справедливо, что их исходный код находится в открытом доступе.
