Huawei облегчила жизнь пользователям без Play Store

21.05.20

Huawei выпустила своё новое приложение - Petal Search. Новое приложение призвано помочь пользователям с поиском приложений для своего телефона.

Petal Search может искать приложения просто по названию и даёт удобный интерфейс для установки. При входе в приложение вас встретит лента новостей от Microsoft.

Приложение позволит скачать Facebook, WhatsApp, Instagram и другие не доступные на данный момент в AppGallery приложения.
