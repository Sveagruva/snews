Microsoft выпустила октябрьское обновление Windows 20H2 с новым меню «Пуск»

22.10.20

Microsoft официально начала постепенное обновление устройств на Windows 10 на новый крупный релиз. Октябрьское обновление представляет несколько нововведений, улучшение безопасности и увеличения стабильности системы.

В новом обновлении Windows, Microsoft изменила меню «Пуск». Это самое заметное изменение, которое было сделано. Иконки и плитки в меню «Пуск» больше не будут окружены акцентным цветом, который выбирает пользователь. Для цвета фона теперь будет только два варианта — оттенки белого и черного.

Приложение настроек и центр уведомлений претерпели небольшие изменения. Приложение «параметры» получило ещё несколько функций, которые раньше были доступны только через приложение «панель управления». А уведомления теперь будут иметь логотип приложения, которое отправила его.

Также теперь при нажатии сочетания клавиш ALT и TAB в появившемся списке открытых окон будут присутствовать вкладки из Microsoft Edge.
