Тинькофф запустил карту с информацией о доступности стран для въезда

19.06.20

Тинькофф выложил на своём сайте карту с названием «Куда лететь?». Карта показывает какие государства, открыли свои границы, а какие ещё нет.

Данные с информацией о доступности стран собираются из открытых источников, например из газет. При наведении мышки на страну она подсветится определённым цветом. Также сверху находятся кнопки для показа стран по доступности.

Для наглядности сервис использует покрас в разные цвета. Зелёный цвет – страна полностью открыта. Жёлтый – въезд возможен с некоторыми ограничениями. Красный – въезд запрещён.

При клике на страну, будет показана более подробная информация со ссылкой на источник. Под логотипом расположилась строка поиска. Поиск можно выполнять по названию страны или города.
