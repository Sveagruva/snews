В Twitter могут появится реакции через эмодзи

12.06.20

Twitter тестирует реакцию на твиты с помощью эмодзи. Их нашла веб-разработчик Джейн Манчун Вонг и рассказала об этом в своём твите:

Реагировать на пост с помощью эмоций стало давно привычным пользователям Facebook. Эмоции должны дать понять реакцию аудитории.

Кроме реакций с эмодзи на скриншоте видна опция «отреагировать с помощью Fleet». Fleet - это своеобразный аналог сторис. Он представляет собой обычный ретвит, который будет удалён автоматически через 24 часа и на который нельзя будет никаким образом отреагировать.
