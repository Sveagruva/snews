Российская таможня создаст новую категорию отправлений для иностранных интернет-магазинов

25.12.20

Российская, государственная таможня планирует создать отдельную категорию специально для почтовых отправлений из иностранных онлайн магазинов.

Данная идея появилась в связи с утроением количества покупок на иностранных онлайн-площадках.

По словам главы ФТС, Владимира Булавина, специально для менеджмента отправлений из этой новой категории будет создана новая организация под названием «оператор интернет-торговли». Данная организация будет отвечать за прямую коммуникацию с крупными интернет площадками и доставку их отправлений.

Помимо этого стало известно об идеи создания таможенного склада, который может быть использован для более эффективной доставки.
