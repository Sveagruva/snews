Новый Microsoft Edge начали распространять через центр обновлений

04.06.20

Недавно Microsoft выпустила очередное проблемное обновление для Windows - 2004. Теперь компания решила также обновить и браузер.

Главное отличие нового Edge от старого является смена движка. Раньше браузер использовал собственную разработку EdgeHTML. Компания вместо разработки собственного движка решила взять готовый - Chromium. Движок для рендеринга страниц будет использоваться такой же, как и у Google Chrome, Blink.

Компания давала возможность протестировать браузер на новом движке просто скачав и установив его. На данный момент в браузере отсутствует синхронизация истории и вкладок, данные функции обещают добавить летом. Также будет добавлена боковая панель с поиском и вертикальные вкладки.
