Tesla пересмотрела ценовую политику

27.05.20

Два месяцев простоя явно не пошли на пользу экономике всех стран. Автопроизводители начали возобновлять своё производство ещё с середины апреля.

Tesla возобновила своё производство только 13 мая нарушив при этом закон. Колоссальное падение продаж Tesla пытается исправить понижением цен на свои электромобили на 6%. Цены на новую Model Y компания не затронула.

В начале 2017 года Tesla отменила бесплатную зарядку для Model 3. Сейчас же компания компания оставит бесплатную зарядку только для Model Y
