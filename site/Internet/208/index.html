Вышло обновление приложения Zoom с поддержкой новых MacBook на процессорах M1

21.12.20

Одно из самых популярных приложений для совершения видеозвонков, Zoom, получило нативную версию для новых MacBook на фирменных процессорах Apple.

До выхода данного обновления пользователи были вынуждены использовать версию написанную для моделей использующих Intel процессоры с помощью Rosetta 2.

По причине ненативности, приложение, по комментариям пользователей, потребляло на 20% больше энергии. Несмотря на это, каких либо значительных багов не было обнаружено.
