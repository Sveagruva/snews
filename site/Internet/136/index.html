Microsoft закрывает стриминговую платформу Mixer

24.06.20

Стриминговая платформа Mixer закрывается из-за того что не оправдала ожиданий компании по количеству зрителей. Об этом она заявила в своём блоге. Привлечение известных англоговорящих стримеров дало большую аудиторию для проекта, который только запустился.

Текущая аудитория Mixer составляет 25 тысяч зрителей, что сравнимо с текущей аудиторией шахмат на Twitch. Стриминговый сервис является очень важной частью бизнеса Microsoft. Поэтому компания заключила соглашение с Facebook. Именно данный сервис будет по умолчанию предоставляться для стриминга игр на Xbox. С 22 июля сервис будет окончательно закрыт, а его сайт будет перенаправлять на Facebook Gaming.
