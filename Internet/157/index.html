Apple приобрела стартап, который может превратить iPhone в платёжные терминалы

02.08.20

Apple купила технологический стартап Mobeewave за $100 млн, который занимался созданием ПО для приёма платежей с телефона.

Для использования телефона в качестве терминала не требуется никакого дополнительного оборудования. Для обмена данными между устройствами будет использоваться чип NFC.

На сайте стартапа в качестве иллюстраций показано как будет совершаться оплата через андроид телефон. Выпустит ли компания приложения для андроида пока неизвестно. Скорее всего Apple захочет сохранить уникальность и запретит использовать технологию на андроид устройствах.

Принятие платежей с помощью телефона позволит избавиться от касс в некоторых магазинах.
