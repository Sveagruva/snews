Apple выпустила iOS 13.5

22.05.20

Apple выпустила в релиз iOS 13.5. Помимо улучшений производительности и исправления багов были внесены новые функции связанные с коронавирусом. Теперь при разблокировке FaceID может определить если пользователь носит маску и сразу предложить ввод пароля.

Также был добавлен API Exposure Notification, который на данный момент не работает из-за отсутствия приложения. Данный API был разработан совместно с Google для того что бы предупредить людей о том, что они контактировали с заражёнными или были поблизости с ними.

Была добавлена настройка в приложении FaceTime. Теперь можно зафиксировать размер всех видеофрагментов при групповом звонке. Ранее фрагмент с активным участником становится больше остальных. iPhone или Apple Watch теперь могут передать медицинскую информацию при экстренных случаях.
