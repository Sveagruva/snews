Huawei представила беспроводные наушники FreeBuds Pro с динамическим шумоподавлением

11.09.20

Huawei представила беспроводные наушники FreeBuds Pro. В качестве киллер фичи наушники обладают динамическим шумоподавлением, которое складывается из активного шумоподавления и системы распознавания шума среды.

Новая система шумоподавления «Dynamic Noise Cancellation» с помощью алгоритмов искусственного интеллекта в реальном времени понимает каким образом нужно удалять посторонние звуки. Также они поддерживают технологию Dual-device connection, которая позволяет подключатся к двум устройствам одновременно.

Наушники работают на процессоре Kirin A1, который компания разработала самостоятельно. В качестве ещё одной особенности, которую Huawei сделали первые — это двойная антенна. Для работы наушники используют протокол Bluetooth версии 5.2.

Автономность наушников составляет 4,5 часов с включенным и 7 часов с выключенным динамическим шумоподавлением. С помощью кейса наушники можно будет зарядить три раза.

В продаже они станут доступны с октября по стоимости в 200 евро.
